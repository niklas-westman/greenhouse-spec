# Greenhouse Rules

This file is human-owned.

Agents may propose changes, but must not silently rewrite authored roots.

## Defaults

- Keep edits scoped to the request.
- Prefer existing repo commands and local patterns.
- Run routed validation before claiming work is done.
- Record skipped or blocked validation explicitly.
- Do not bulk-read `.greenhouse/evidence/`; use `.greenhouse/grown/evidence-index.yaml` and read only specific evidence when relevant.
